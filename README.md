# 🔙 E-Commerce Backend

Backend API for the e-commerce app, built with **Node.js**, **Express**, and **MongoDB**.

## 🧩 Features

- RESTful API
- User registration/login
- Product CRUD
- Cart management per user

## ⚙️ Tech Stack

- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- multer
- cloudinary

## 📦 Run Locally

```bash
cd backend
npm install
npm run dev
